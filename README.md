# webpack-playground
持续添加项目中的 Webpack 最佳配置......

### buil playground
```bat
# build with development mode
npm run build:dev

# build with production mode
npm run build:prod

# build the static assets
npm run build:dll

# server to verify assets cache
npm run test-server
```
